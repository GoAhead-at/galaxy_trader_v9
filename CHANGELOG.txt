========================================


Version 0.12.4 (2026-04-04)
FEATURE: MK1 new order parameter "Cargo/Profit" (`sellselectionmode`) to switch between profit-first and cargo-fill-first sell selection.
CHANGE: MK1 cargo mode now prioritizes highest station ware fill for primary and secondary ware selection (score tie-break).
CHANGE: MK1 cargo mode fill % is reservation-aware (`get_ware_reservation` type buy on home station): effective count excludes other ships' reservations so heavily reserved wares are deprioritized vs. equally full but unreserved stock.
CHANGE: MK1 now prefers a single full-cargo trade run over multi-ware chaining when a valid single run can satisfy selltarget.
FIX: MK1 multi-ware selection now enforces one shared remaining cargo-volume budget across secondary wares.
FIX: MK1 queued trade creation switched to phased queueing to avoid alternating buy/sell mini-runs while preserving load-before-sell execution safety.
FIX: MK1 single full-cargo preference uses queueable clamped amounts (offer/capacity-aware), preventing over-100% fill mis-selection.
FIX: Ship status labels keep `TR` during active training; blocked label logic remains separate.
FIX: Pilot redistribution excludes blocked pilots and ships already in active/queued pilot-exchange orders.
FIX: Pilot exchange is now session-driven and save/load persistent (`global.$GT_PilotExchangeSessions` / `...ByShip`) with reliable swap trigger and cleanup.
FIX: Captain swap execution resolves ships by authoritative idcode/UID and applies idempotent pair execution.
FIX: Abort/fail/release handling is race-hardened with delayed aborted-session recheck before terminal cleanup.
FIX: Ship mod reconciliation keeps already-correct GT mods in place (no unnecessary dismantle/reinstall).
REFACTOR: Removed legacy swap-tracker scaffolding; kept one authoritative MD session pipeline.
REFACTOR: Centralized PE state checks/name-refresh triggers in `md.GT_Ship_State_Utilities`.
CHANGE: Fleet commander swaps only when receiving a same-class pilot from a higher penalty tier.
